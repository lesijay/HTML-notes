What is HTML

HTML stands for Hypertext Markup Language.HTML documents are the building block of the web.It is made up of texts that the user will actually read in their browser.

The Web

The Web (or World Wide Web) is a collection of documents and other files. Most of the documents are made using Hypertext Markup Language (HTML).

Markup

Markup is used in HTML to tell the browser how the text should look like. Most of the markups contain an opening tag and a closing tag.The tag enclosed in the less than and greater than symbol is called an opening tag and when the "/" symbol is placed after the less than sign, it is called a closing tag. The whole write up is called an element

Whitespace is a term programmers use to describe: spaces, tabs and line breaks.
The line break tag "br" enclosed within the less than and greater than symbol is used to indicate a line break in poems.It doesn't have a closing tag, this is because it has no element in it. It is thereby called a "void or empty element"

Nested elements mean one element going inside another. Note that elements can not be overlapped and a paragraph cannot be nested into another. The h1 element can be used for headlines, or section headings. There are actually six of these heading elements: h1 through h6

Block and Inline Elements

The paragraph element is a block element, which means the browser creates a box around it. It also generates a margin around this box, to visually separate the paragraph from surrounding elements. In contrast, br is an inline element that just creates a line break. It doesn't have a box around it and doesn't create any margin.

The div is a short form for division element.It can be used to divide a page into different sections. It is a block element like the p element, but while a p element is specifically meant to contain text, the div element is a generic container for whatever other elements you might want to put inside. You can use the div element to organize the content and divide the page into sections.

DOCTYPE Declaration: Document Type declaration The (!DOCTYPE) must be the very first thing in a HTML document before the (html) tag. It is not an HTML tag; it is an instruction to the web browser about what type of document to expect.
HTML tag: The (html) tag tells the browser that this is an HTML document. It is the container for all other HTML elements (except for the !DOCTYPE tag).
HTML head tag: The (head) element is a container for all the head elements.It can include a title for the document, scripts, styles, meta information, and more.
title element: This element is required in an HTML document. It tells the browser the name tobe displayed inthe browser tab.
body element:The (body) tag defines the document's body. It contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
